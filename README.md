# IMLSSR
This is the official code of our paper "Stereoscopic Image Super-Resolution with Interactive Memory Learning"

## Requirements
- Python 3.6 (Anaconda is recommended)
- skimage
- imageio
- Pytorch 1.4.0
- torchvision  0.5.0
- tqdm 
- pandas
- cv2 (pip install opencv-python)


### Test
```bash
python test.py 
```

## Train
```bash
python train.py 
```
